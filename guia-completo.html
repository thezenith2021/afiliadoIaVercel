Passo 1 — GitHub + Vercel

Subir os arquivos e deixar o app online

~30 min
📥

Baixar e extrair o ZIP

Role para cima nesta conversa → encontre o arquivo afiliadoai-vercel.zip → clique em Download

Depois: Vá na pasta de Downloads do seu celular/computador → pressione e segure o ZIP → Extrair aqui → vai aparecer a pasta afiliadoai
Baixei o arquivo afiliadoai-vercel.zip
Extraí o ZIP e tenho a pasta afiliadoai
1

Criar repositório no GitHub

Acesse o GitHub e crie um repositório para guardar os arquivos do app

🐙 Abrir GitHub → New Repository
Preencha assim:
• Repository name: afiliadoai
• Visibilidade: Public
• Clique no botão verde Create repository
Repositório afiliadoai criado no GitHub ✅
2

Subir todos os arquivos

Na página do repositório criado, clique no link azul "uploading an existing file"

Abra a pasta afiliadoai e arraste:
📁 Pasta src/ (App.jsx, firebase.js, etc)
📁 Pasta public/ (index.html)
📄 package.json
📄 vercel.json
📄 .gitignore
📄 README.md
⚠️ Importante: Arraste TANTO as pastas quanto os arquivos soltos. Se esquecer algum o app pode não funcionar.

Depois role a página para baixo e clique em Commit changes

Todos os arquivos enviados ao GitHub ✅
Cliquei em "Commit changes" ✅
3

Importar no Vercel

Abra o Vercel e importe o repositório que você acabou de criar

🚀 Abrir Vercel → New Project
Passos:
1. Clique em Continue with GitHub
2. Encontre o repositório afiliadoai → clique em Import
3. NÃO clique em Deploy ainda!
4. Primeiro configure as variáveis no Passo 4
Projeto importado no Vercel ✅
🔥

Passo 2 — Firebase

Banco de dados para salvar cliques, vendas e usuários

~45 min
1

Criar projeto no Firebase

🔥 Abrir Firebase Console
Passos:
1. Clique em "Criar um projeto"
2. Nome do projeto: afiliadoai
3. Clique em Continuar
4. Desative o Google Analytics (opcional)
5. Clique em Criar projeto
6. Aguarde ~1 minuto ☕ e clique em Continuar
Projeto afiliadoai criado no Firebase ✅
2

Ativar Firestore Database

Passos:
1. No menu esquerdo clique em Firestore Database
2. Clique em "Criar banco de dados"
3. Selecione "Iniciar no modo de teste" → Avançar
4. Região: selecione southamerica-east1 (São Paulo)
5. Clique em Ativar e aguarde
Firestore Database ativado em São Paulo ✅
3

Ativar autenticação anônima

Passos:
1. Menu esquerdo → clique em Authentication
2. Clique em "Primeiros passos"
3. Na lista, clique em Anônimo
4. Ative a chave → clique em Salvar
Autenticação Anônima ativada ✅
4

Pegar as credenciais do Firebase

Passos:
1. Clique na ⚙️ engrenagem (topo esquerdo) → Configurações do projeto
2. Role a página até "Seus apps"
3. Clique no ícone </> (Web)
4. Apelido do app: afiliadoai-web
5. Clique em Registrar app
6. Vai aparecer um bloco de código — copie todos os valores
⚠️ Guarde bem! Você vai precisar desses valores no Passo 4. Deixe a página aberta ou copie num bloco de notas.
As credenciais vão parecer assim:
apiKey: "AIzaSy..."
authDomain: "afiliadoai.firebaseapp.com"
projectId: "afiliadoai-208b7"
storageBucket: "afiliadoai.appspot.com"
messagingSenderId: "123456789"
appId: "1:123:web:abc"
Credenciais do Firebase copiadas ✅
🛒

Passo 3 — API Mercado Livre

Para buscar produtos reais com preço e imagem

~1 hora
1

Acessar o portal de desenvolvedores

🛒 Abrir ML Developers
Passos:
1. Clique em "Entrar" no canto superior direito
2. Use sua conta do Mercado Livre para logar
3. Se não tiver conta, crie uma em mercadolivre.com.br — é grátis
Logado no portal de desenvolvedores ML ✅
2

Criar sua aplicação

Preencha os campos assim:
• Nome: AfiliadoAI
• Descrição: App de afiliados com IA
• URL do site: https://afiliadoai.vercel.app
• URL de redirect: https://afiliadoai.vercel.app
• Scopes: marque read_catalog e public_info
• Clique em Salvar
Aplicação AfiliadoAI criada no ML ✅
3

Gerar o Access Token

Passos:
1. Na página da sua aplicação, procure o botão "Gerar Token de Teste"
2. Clique nele
3. Um token vai aparecer no formato:
APP_USR-1234567890-abcdef...
4. Copie e guarde esse token
⚠️ Atenção: O token de teste expira em 6 horas. Para uso contínuo, implemente o fluxo OAuth do ML — há documentação em developers.mercadolivre.com.br
Access Token do ML copiado ✅
🎬

API de Vídeo Gratuita — JSON2Video

Para criar vídeos automáticos dos produtos sem pagar nada

🎬 Pegar API Key Grátis
Passos:
1. Acesse json2video.com/get-api-key
2. Digite seu e-mail
3. Clique em "Get my free API Key"
4. Verifique seu e-mail — a key chega em minutos
5. Copie e guarde a key
✅ 100% grátis! Sem cartão de crédito. Inclui 600 segundos de vídeo para criar seus primeiros vídeos dos produtos.
API Key do JSON2Video recebida no e-mail ✅

Fazer o Deploy final 🚀

Passos:
1. Após adicionar as variáveis, clique em Deployments no menu

5. Quando ficar verde aparece seu link!
🎉 Resultado: Seu app vai estar online em:
https://afiliadoai.vercel.app
